Параметры для рассчета

Введите, пожалуйста, адрес вашего вебсайта. Адрес может содержать www. Затем, через точку, название сайта и расширение, которое обозначает страну или типа сайта. Например: ru – Россия, ua – Украина, com – международные сайты, org – некоммерческие. Два одинаковых адреса с разными расширениями могут вести на два разных сайта.

Проанализировав ваш вебсайт, специалисты Gravitec.net смогут провести для вас развёрнутый анализ сайта с точки зрения канала push уведомлений.

А именно, спрогнозировать:

  • рост вашей базы push подписчиков;
  • объем уведомлений, которые вы сможете доставлять вашим подписчикам по мере сбора базы подписчиков;
  • количество переходов на ваш сайт по разосланным вами push уведомлениям;
  • средний месячный прирост посещаемости при регулярных push рассылках и прочее;
  • среднюю доходность канала push уведомлений, исходя из данных конверсии в покупку и доходности на одного покупателя.

Это показатель общей посещаемости НЕ уникальных пользователей сайта в месяц. Можно взять из открытых источников, например, www.similarweb.com

На основании данных о посещаемости калькулятор высчитает основные показатели эффективности канала push уведомлений.

Средняя конверсия подписки на push уведомления составляет от 0,5% до 9,0% от неуникальных посещений вашего вебсайта.

Конверсия может показаться невысокой, но это 0,5% - 9,0% лояльных посетителей, которые дали разрешение на отправку им уведомлений от сайта. Push уведомления работают именно с этой лояльной частью аудитории.

http/https:*

Протокол вашего вебсайта. От протокола вашего сайта напрямую зависит конверсия подписки на push уведомления.

HTTPS (аббр. от англ. HyperText Transfer Protocol Secure) — расширение протокола HTTP для поддержки шифрования данных в целях повышения безопасности.

Так как Google является адептом защищённой передачи данных, технология push изначально была разработана для https сайтов.

Сервис Gravtec.net предоставляет возможность http сайтам использовать канал push уведомлений. В этом случае подписка происходит в два клика и это влияет на конверсию подписки в сторону её снижения.

Если вы используете http протокол на вашем сайте, Gravitec.net и Google рекомендуют вам задуматься о переходе на Secure протокол https.

Кроме явных преимуществ https технологии (шифрование личных, финансовых данных ваших клиентов, повышение степени доверия посетителей к вашему вебсайту и проч.), конверсия подписки на push уведомления вырастает в 5-7 раз по сравнению с http сайтами.

Частота рассылок push уведомлений подписчикам.

Влияет на лояльность аудитории к push уведомлениям, на эффективность каждой отдельной рассылки и общие показатели эффективности данного инструмента. Зависит от тематики уведомлений, рынка, на котором работает компания.

При редких, не регулярных рассылках эффективность push канала будет минимальной.Очень частыми рассылками можно спровоцировать высокий % отписки.

Важно определить оптимальную частоту, с которой аудитория вашего сайта готова воспринимать ваши рассылки. И конечно не стоит забывать о том, что база подписчиков - это живые люди и контент push уведомлений должен быть интересен вашей аудитории.

CTR (от англ. click-through rate — показатель кликабельности) - метрика в интернет-маркетинге. CTR определяется как отношение количества кликов по push уведомлениям к количеству уведомлений доставленных подписчикам, измеряется в процентах.

Показатель зависит от правильного использования данного канала коммуникации. Может быть увеличен, благодаря использованию рекомендаций специалистов Gravitec.net, основанных на статистике работы более чем с 2500 сайтов.

На показатель CTR в первую очередь влияют:

  • Привлекательность пуша (иконка, заголовок, полезный/интересный/важный контент и т.п.);
  • Дополнительные интерактивные элементы push уведомления (большая картинка - 360х240, кнопки - каждой из которых может соответствовать отдельная ссылка);
  • Время отображения уведомления (время в течение которого push уведомление показывается на мониторе подписчика, повышает вероятность того, что подписчик увидит уведомление и перейдёт по нему).

Уровень конверсии (conversion rate) — это процентное соотношение посетителей магазина, сайта, которые совершили выбор, осуществили покупку или другое целевое действие, к общему числу посетителей, перешедших на сайт по push уведомлению (рассылке).

Показатель лояльности ваших подписчиков и коммерческой эффективности ваших рассылок.

Конверсия в покупку в основном зависит от своевременности и ценности вашего коммерческого предложения для ваших подписчиков.

Выбор правильных параметров рассылки коммерческого push уведомления может существенно повысить эффективность ваших push кампаний.
